## 8️⃣ CLEAN TEXT BEFORE SENDING (VERY IMPORTANT)

Add preprocessing:

```ts
const cleanText = text
  .replace(/\b(CONTACT|SKILLS|EDUCATION|EXPERIENCE|PROFILE|SUMMARY|LANGUAGES|CERTIFICATIONS)\b/gi, "")
  .replace(/\n{2,}/g, "\n")
  .replace(/\s{2,}/g, " ")
```

---

## 9️⃣ PARSE RESPONSE SAFELY

```ts
const result = await response.json()

const toolCall = result?.choices?.[0]?.message?.tool_calls?.[0]

if (!toolCall?.function?.arguments) {
  throw new Error("AI did not return structured data")
}

const parsedData = JSON.parse(toolCall.function.arguments)
```

---

## 🔟 POST-VALIDATION (CRITICAL)

Fix bad values:

```ts
const invalid = ["CONTACT", "SKILLS", "EDUCATION", "EXPERIENCE"]

if (invalid.includes(parsedData.personalInfo.firstName?.toUpperCase())) {
  parsedData.personalInfo.firstName = ""
}

if (invalid.includes(parsedData.personalInfo.jobTitle?.toUpperCase())) {
  parsedData.personalInfo.jobTitle = ""
}
```

## 1️⃣1️⃣ FALLBACK EXTRACTION (IMPORTANT)

if (!parsedData.personalInfo.email) {
